# To-Do List (React + Vite)

A small single-page application for managing a to-do list, built with React and Vite.

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS Modules

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All / Active / Completed)
- Button hover and active animations
- Animated task removal
- Responsive layout

## Installation & Run

1. Clone the repository:

git clone https://github.com/SlavaCHPS/todo-app.git

2. Install dependencies:

npm install

3.Run the project in development mode:

npm run dev
